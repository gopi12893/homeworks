INHERITED SCOPE

a child scope that is within the parent scope
all parent scope members can be accessed by child scope

EXAMPLE

parent scope, controller

controller1: a={{a}}, b={{b}}
modify a=

if we modify here there is also a modification in the child 'a' automatically,
because it is a inherited child

inherited scope(child) is included here

it have its own values c,d and also accesses parent values a,b

controller1: a={{a}}, b={{b}}
controller2: c={{c}}, d={{d}}
modify a=

if we modify parent value in child here there is only modification in of a in child only
